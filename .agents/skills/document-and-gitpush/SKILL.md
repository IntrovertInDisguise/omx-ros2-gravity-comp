# Document and Git Push
This skill allows you to document your plans, changelog, and current status, and then git push to a specified branch in GitHub with an appropriate commit message. 
## Usage
To use this skill, simply provide the necessary information in the following format:

```Document and Git Push:
- Plans: [Your plans here]
- Changelog: [Your changelog here]
- Current Status: [Your current status here]
- GitHub Repository: [Your GitHub repository URL here]
- Branch: [The branch you want to push to here]
- Commit Message: [Your commit message here]
```
## Example
```Document and Git Push:
- Plans: Implement new feature X by the end of the month.
- Changelog: Added new feature X and fixed bug Y.
- Current Status: Feature X is complete and tested.
- GitHub Repository: https://github.com/user/repo.git
- Branch: main
- Commit Message: Implement new feature X and fix bug Y
```
## Example
```Document and Git Push:
- Plans: Refactor codebase for better performance.
- Changelog: Refactored codebase and improved performance by 20%.
- Current Status: Refactoring is complete and performance has been improved.
- GitHub Repository: https://github.com/user/repo.git
- Branch: main
- Commit Message: Refactor codebase and improve performance
```
