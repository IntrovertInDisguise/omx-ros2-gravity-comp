This skill is designed to assist in continuing ongoing tests and diagnosing issues when they arise. If you find yourself stuck during testing, this skill will help you narrow down the problem and provide a concise LLM dump that outlines the issue clearly and exhaustively, without overwhelming the context window of the LLM.
### How to Use This Skill
1. **Continue Ongoing Tests**: If you are in the middle of a test and need assistance, simply invoke this skill to get guidance on how to proceed with the next steps in your testing process.
2. **Diagnose Issues**: If you encounter a problem during testing, use this skill to help identify the root cause. It will guide you through a series of questions and checks to narrow down the potential issues.
3. **Generate LLM Dump**: Once the issue is identified, this skill will help you create a concise and comprehensive LLM dump that summarizes the problem. This dump will be designed to provide enough information for an LLM to understand the issue without exceeding the context window limits.
### Benefits of Using This Skill
- **Efficiency**: Quickly continue tests without having to start from scratch, saving time and effort.
- **Problem-Solving**: Systematically narrow down issues to find the root cause, making it easier to resolve problems effectively.
- **Clear Communication**: Generate clear and concise LLM dumps that facilitate better understanding and communication of issues, leading to faster resolutions.
### Conclusion
This skill is an essential tool for anyone involved in testing, providing a structured approach to continue tests and diagnose issues effectively. By using this skill, you can ensure that you are making the most of your testing efforts and efficiently resolving any problems that arise.
